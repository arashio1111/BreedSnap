UPET|CREATE TABLE UPET
(
PNAME TEXT PRIMARY KEY NOT NULL,
SNAME TEXT NOT NULL,
FIRST REAL,
SECOND REAL,
THIRD REAL, 
UPETPICTURE BLOB,
FOREIGN KEY (SNAME) REFERENCES SPECIES(SNAME)
)

SPECIES|CREATE TABLE SPECIES
(
SNAME TEXT PRIMARY KEY NOT NULL,
PERSONALITY TEXT,
HOBBY TEXT,
EYESNOTE TEXT,
MOUTHNOTE TEXT,
NOSENOTE TEXT,
SKINNOTE TEXT,
EATNOTE TEXT,
LIMBSNOTE TEXT,
ACTNOTE TEXT,
EARNOTE TEXT,
OFFICIALPICTURE BLOB)


UPET 表是用户保存自己的宠物的位置： PNAME : 宠物名字 
				   SNAME : 品种
				   FIRST : 模型预测的最高概率品种的概率值
		 		 UPETPICTURE : 用户在预测品种时使用的图片
				 ( SECOND, THIRD 这两个column将会删除掉）
SPECIES 表是支持预测的所有品种      ：SNAME : 品种
			         PERSONALLITY : 性格
				  HOBBY : 爱好
				 以下各种NOTE : 相关部位或方面的饲养建议或注意点
				 OFFICIALPICTURE : 之后将会保存官方的品种相片
